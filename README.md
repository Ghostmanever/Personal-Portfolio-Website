# Aryan Kaushal Portfolio

This is a clean, dark-themed personal portfolio website built with HTML and CSS only.

## Features
- Home page with a welcome introduction
- Navigation bar linking to About Me page
- About Me page with personal info
- Consistent dark theme
- Copyright footer

## Usage
Open `index.html` in your browser to view the site.

## Customization
Edit `about.html` to add more personal details or update the style in `style.css`.
